A  creative personal profile card built with React and TypeScript.

## Technologies Used 
* **React** (UI Library)
* **Vite** (Build Tool)
* **TypeScript** (Programming Language)
* **CSS** (Styling)
* **React Icons** (Social Media Icons)

## How to Run 

**1. Install dependencies:**
Open your terminal inside the project folder (`react-training`) and run:
`npm install` 

**2. Start the development server:**
Once the installation is complete, start the project by running:
`npm run dev`
